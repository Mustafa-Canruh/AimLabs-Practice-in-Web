# Pro Labs: Aim Trainer 🎯

A free, fast, and responsive 2D Aim Trainer built entirely for the browser. Practice your reflexes and compete with others without downloading any heavy files or applications.

## 🎮 Live Demo & How to Play

**You don't need to download any files, install any code editors, or run terminal commands!**

Simply click the link below to play directly in your browser:
👉 **[Play Pro Labs Aim Trainer Here](https://mustafa-canruh.github.io/AimLabs-Practice-in-Web/)**

*(Note: The game works entirely in the browser. No installation required!)*

### 🚀 Features
* **Zero Installation:** 100% web-based HTML/CSS/JS game.
* **Global Leaderboard:** Compete with friends for the highest score (Powered by Firebase).
* **Real-time Stats:** Track your accuracy, score, and personal bests.
* **Multiple Modes:** Choose between 15s, 30s, or 60s rounds.
* **Clean UI:** Distraction-free, dark-mode focused design.

### 🛠️ Built With
* HTML5, CSS3, JavaScript
* Firebase Firestore (Backend / Leaderboard)
